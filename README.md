### GoShelf
Simple project for Mobile Aplications class at uni for tracking your home library, written (backend at least) in Golang.

#### Technologies used
* Tesseract (ocr)
* Postgresdb 
* ZeroTier (vpn - I got no public ip)
* Docker (hell yeah)
* Golang (backend)
* Java/Kotlin (yuck) (frontent)

Work very much in progress...
